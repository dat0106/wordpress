== Description ==

Adds widget which you can use to add Facebook page feed and display it on your Wordpress site.

== Installation ==

1. Upload fbpage-feed to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to 'Appearance > Widgets' to add and configure your widget

== Frequently Asked Questions ==

= How do I find out my Facebook page ID? =

You can use: http://findmyfacebookid.com
